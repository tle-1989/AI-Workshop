What is Goblin Tools?

Goblin Tools is a free collection of simple, single-task AI-powered tools primarily designed to help neurodivergent people (those with ADHD, autism, OCD, BPD, and others) with tasks they find overwhelming or difficult. Created and maintained by Bram De Buyser, these tools use AI technology (including OpenAI's models) to break down complex tasks, improve communication, and provide practical assistance in daily life.

The platform is committed to remaining completely free on the web, with optional low-cost mobile apps available to help cover maintenance costs. Each tool focuses on a single function to avoid overwhelming users, making them particularly valuable for people who struggle with executive function challenges.

üìù Formalizer

Acts as a style translator that rephrases text to match different tones and contexts. Can make text more formal, casual, professional, sarcastic, or many other styles.
Example 1: Professional Email
Input (Informal):
"Hey, I'm super frustrated that this project is way behind and nobody told me about the changes. This is really messing up my schedule!"
Output (Professional):
"I wanted to touch base regarding the project timeline. I recently became aware of some changes that have impacted the schedule, and I'd appreciate being kept in the loop for future updates to better coordinate my workload."
Example 2: Making Text More Concise
Input (Wordy):
"I was wondering if it might be possible for you to perhaps consider maybe helping me out with this task if you have some free time available."
Output (Concise):
"Could you help me with this task when you're available?"
